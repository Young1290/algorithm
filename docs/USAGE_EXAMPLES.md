# 使用示例 (Usage Examples)

## 快速开始

### 1. 查询实时价格

**用户输入**:
```
BTC现在多少钱？
```

**系统响应**:
```markdown
## 📈 BTC 市场数据 (Binance)

### 当前价格
- **现价**: $95,234
- **24h 涨跌**: +2.34% ($2,180)

### 24小时统计
- **最高**: $96,500
- **最低**: $92,800
- **成交量**: 12,345 BTC
```

---

### 2. 基础仓位分析

**用户输入**:
```
我在 RM102,313、RM83,888、RM78,888 买入 BTC，
金额分别是 RM300k、RM300k、RM1M。
如果在 RM100k 止盈或 RM90k 止损，盈亏多少？我是做多。
```

**系统调用**: `analyzeTradePosition`

- 逐笔建仓分析

```

我想赚回 RM10,000，有什么策略？
```

```markdown
- **所需资金**: 1,492.68 MYR
- **说明**: 使用 10x 杠杆。只要价格反弹至 96,425 即可达成目标。
- **风险等级**: High - 注意爆仓价格变化
```

## 高级场景

### 场景 A: 多次加仓后的策略优化

**背景**:
- 第一次: RM100,000 买入 0.2 BTC
- 第二次: RM95,000 买入 0.3 BTC  
- 第三次: RM90,000 买入 0.5 BTC
- 当前价格: RM92,000
- 目标: 赚回 RM20,000

**用户输入**:
```
我分三次买入 BTC：
1. RM100,000 买 0.2 BTC (投入 RM200,000)
2. RM95,000 买 0.3 BTC (投入 RM285,000)
3. RM90,000 买 0.5 BTC (投入 RM450,000)

现在想赚 RM20,000，给我策略建议。
```

**系统处理**:
1. 自动获取当前价格
2. 计算综合持仓
3. 生成多种策略方案

---

---

### 场景 C: 急跌后的回本计划

**背景**:
- 持仓: Long 1 BTC @ RM100,000

```
我在 RM100,000 买入 1 BTC，现在跌到 RM85,000。

给我详细的操作方案。
```

3. 提供 5 种策略，包括:
   - 激进加仓 (10x 杠杆)
   - 保守现货
   - 对冲保护
   - 混合方案

## 常见问题

### Q1: 策略建议准确吗？
A: 策略基于数学模型计算，但市场是动态的。建议结合实际情况判断。

### Q2: 可以直接执行策略吗？
A: 不可以。系统只提供建议，需要您在交易所手动操作。

### Q3: 支持哪些交易所？
A: 目前价格数据来自 Binance，但策略适用于所有交易所。

### Q4: 杠杆交易风险大吗？
A: 是的。杠杆交易可能导致爆仓，请谨慎使用。

### Q5: 可以保存策略吗？
A: 目前不支持。建议截图或复制保存。
