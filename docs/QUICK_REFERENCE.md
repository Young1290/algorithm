# 快速参考 (Quick Reference)

## 🎯 常用命令

### 查询价格
```
BTC现在多少钱？
给我看看ETH的价格
SOL当前价格是多少？
```

### 仓位分析
```
我在 RM100,000 买入 0.5 BTC，如果在 RM110,000 卖出，赚多少？
```

### 策略规划
```
我持有 0.5 BTC，均价 RM90,000，想赚 RM10,000，给我策略
```

---

## 🛠️ 工具列表

| 工具名称 | 功能 | 使用场景 |
|---------|------|---------|
| `getBinanceMarketData` | 获取实时价格和市场数据 | 查询当前价格、24h统计 |
| `planToAchieveProfitTarget` | 生成达成盈利目标的策略 | 制定回本/盈利计划 |
| `analyzeTradePosition` | 分析持仓盈亏 | 计算当前盈亏、止盈止损 |
| `calculateTargetPrices` | 计算目标价格 | 知道要涨/跌到多少才能达成目标 |
| `suggestPositionAdjustment` | 仓位调整建议 | 对冲、加仓建议 |

## 📊 策略速查

| 策略 | 风险 | 资金效率 | 适用场景 |
|-----|------|---------|---------|
| 🔥 10x杠杆加仓 | 高 | 最高 | 确信反弹 |
| 🛡️ 买入现货 | 低 | 低 | 长期看好 |
| ⚖️ 对冲策略 | 中 | 中 | 不确定方向 |
| 🍹 混合策略 | 中 | 中 | 平衡风险 |
| 🎯 止盈建议 | 低 | - | 接近目标 |

## 💡 使用技巧

### 1. 不知道价格时
直接说持仓信息，系统会自动获取当前价格

### 2. 多次加仓
列出所有买入记录，系统会计算综合持仓

### 3. 对比策略
要求系统分别展示不同策略的效果

### 4. 设置限制
说明最大可用资金，系统会筛选可行方案

## 🔢 参数说明

### 持仓方向
- `long` / `做多` - 买入看涨
- `short` / `做空` - 卖出看跌

### 杠杆倍数
- `1x` - 现货（无杠杆）
- `10x` - 10倍杠杆（常用）
- `20x` - 20倍杠杆（高风险）

### 保守模式
- `true` - 保守（默认）
- `false` - 激进

## 🚨 常见错误

### 错误 1: API Key 未配置
```
Error: DEEPSEEK_API_KEY is not configured
解决: 在 .env 文件中添加 API key
```

### 错误 2: 无法获取价格
```
Error: 无法获取实时价格
解决: 检查网络连接或手动提供价格
```

### 错误 3: 参数错误
```
Error: Invalid parameters
解决: 检查数值是否为正数，方向是否正确
```
